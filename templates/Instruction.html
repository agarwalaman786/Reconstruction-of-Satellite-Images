Satellite Image Reconstruction



How To Download the Processed Image again

To Download the processed Image type the following URL in the New Tab:

<Main Address>/downloaded_file/<your filename>.jpg

For Eg: http://0.0.0.0:8080/downloaded_file/xyz.jpg



How To see the current state of the uploaded Image

To see the current patch of the image which is being processed type the following URL in the New Tab:

<Main Address>/Gif/<your filename>.gif

For Eg: http://0.0.0.0:8080/Gif/xyz.gif
